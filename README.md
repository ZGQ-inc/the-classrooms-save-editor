# The Classrooms 存档编辑器

本项目是一个基于 Web 的纯静态 GUI 存档编辑器，专为游戏 *The Classrooms* 设计。

![preview](https://the-classrooms.zgqinc.gq/preview.png)

# 功能列表

### 核心功能

* 支持导入导出为 `.json` 或 `.reg` 格式。
* **JSON支持**：支持加载标准的 `.json` 存档文件。
* **注册表解析**：内置强大的 `.reg` 文件解析器，能够读取 Windows 注册表导出的存档，自动识别编码（UTF-16LE）并将 Hex 数据转换为可编辑的 JSON 对象。
* **多选处理**：支持同时选择2个模式存档文件或整个文件夹。
* **双模式并行编辑**：界面采用分栏设计，可同时查看和编辑“Auto Save”和“Non Linear”的存档数据。

### 玩家属性

* **基础资源修改**：
* 修改奖票数量。
* **手电筒增强**：可视化滑块调整手电筒电量（0-240）。
* **背包扩容**：调整背包等级（Level 1 - Level 3）。

* **无尽模式专属**：
* 修改剩余生命值。
* **难度选项**：一键开启“无限生命”和“可获得生命”开关。
* 修改当前深度层级。

### 关卡解锁

* **Auto Save**：
* **磁带收集**：解锁关卡。
* **一键通关**：一键解锁全部磁带。

* **Non Linear**：
* 自定义各类型房间的访问数量。

### 物品栏编辑器

* **分类管理**：提供四个分类标签：可收集、钥匙、钥匙卡、档案。
* **网格交互系统**：
* **直观展示**：物品以 3*5 网格显示，和游戏内布局一致。
* **添加物品**：点击空槽位弹出物品列表，支持**关键词搜索**所有可用物品并添加到背包。
* **修改物品**：点击已有物品可修改数量、替换为其他物品或删除。

### 档案与图鉴预览

* **档案系统**：查看已收集的档案列表。
* **一键解锁**：快速解锁游戏内所有档案条目。
* **实体图鉴**：内置查看器，可预览所有实体的立绘。

* **图片查看器**：
* 点击条目全屏查看档案图片。
* 支持缩放、拖拽平移图片。

### UI/UX

* **i18n**：内置中英文双语切换，实时更新界面文本。
* **响应式设计**：适配桌面端和移动端布局。
* **视觉反馈**：
* 和游戏风格呼应的暗黑风格主题。
* 美化的 Toast 提示框。
* 徽章显示当前文件格式（JSON/REG）。

### 技术特性

* **纯静态**：无需后端服务器，所有数据处理在浏览器本地完成。
* **版本控制**：通过 `version.txt` 自动检测游戏版本兼容性。
